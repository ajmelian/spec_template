- **Claridad funcional:** cada feature debe resolver un problema concreto.

- **Mantenibilidad:** se prioriza codigo simple, tipado, testeable y documentado.
- **Compatibilidad:** las APIs publicas se versionan y no se rompen sin plan de migracion.
